Bloodlines Helper v0.3.0

New Features
- Added element support for account profiles.
- Added account selector display options for elements.

Backwards Macro Updates
- Simplified the Backwards macro settings.
- Removed the macro method selector.
- Added S-release behavior while W is held:
- Holding W + S starts the macro.

Account Manager
- Added more element tracking to account profiles.
- Added support for showing elements in the account selector.
- Fixed a UI issue where unused "Last used" and "Default PlaceId" labels could appear stuck in the top-left of the Account Manager.

Settings
- Added a separate Q Macro toggle under Backwards.
- Cleaned up macro settings by removing unused macro method options.
- Backwards and Q Macro states are now saved and restored through settings.

Cleanup
- Removed old macro test modes.
- Removed unused Account Manager labels.
- Improved macro key release behavior to prevent stuck keys.
- Multiple bug fixes and cleanups
