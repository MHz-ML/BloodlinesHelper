Bloodlines Helper - Update 0.4.0

New Features:
- Added a new Game Info tab.
- Added an Armors section inside Game Info.
- Added armor detail popups with Normal / Ascended stat switching.
- Added an armor comparison tool for comparing two armors side-by-side.
- Added cleaner armor stat formatting for easier reading.

Changes:
- Removed the old Macros tab completely.
- Removed macro hotkey toggling/background macro behavior.
- Kept the Backwards feature available in Settings.
- Cleaned up the sidebar by removing the bottom status card.
- Improved the Game Info layout with a simpler sub-tab style.
- Moved the Compare Armors button into the Armors section.
- Removed unsupported/excluded armor entries from the public release armor list.

Clan Info:
- Clan/Bloodline guide work is still ongoing.
- Existing clan info remains available.
- More clan passives, progression steps, awakenings, and tips will be added over time.
- The Clan Info tab is still actively being expanded as more reliable data is collected.

Notes:
- This update focuses on cleaning up the app for release, removing unused macro systems, and starting the new Game Info database.
- Armor data may continue to be adjusted as balance changes or more information becomes available.
